# Tilt Type

> A typographic experiment. Tilt your device to feel gravity pull the words apart.

Live demo: https://xiaokunduan.github.io/tilt-type

## Features

- **Gravity simulation** — Each character is affected by device orientation or mouse movement
- **Physics-based** — Spring forces + gravity vectors make letters drift apart naturally
- **Mobile support** — Device Orientation API (accelerometer) for phone tilt
- **Desktop support** — Mouse position simulates gravity direction
- **Custom text** — Type any text and press Enter or click Render

## How it works

1. Open `index.html` in a browser
2. On mobile: tap the canvas, then tilt your phone
3. On desktop: move your mouse to pull the letters

## Tech

- Pure vanilla JS + HTML5 Canvas
- No dependencies
- ~10KB single file
